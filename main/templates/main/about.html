About MGM EDUZONE

MGM EDUZONE is an Online Notes Management System designed to help students easily access academic study materials. Students can view and download notes organized by semester, subject, and module.

The platform simplifies the process of sharing study materials and ensures that students can quickly find the resources they need for their academic preparation.

Technologies Used

  • HTML
  • CSS
  • Python
  • Django Framework
  • SQLite Database

Key Features

  • Semester-wise notes organization
  • Module-based study materials
  • Quick search for subjects
  • Easy note download
  • Admin panel for uploading and managing notes
